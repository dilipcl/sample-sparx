Scenario Type
Create order Basic Path
Details:
1. Use case begins when Storeroom worker selects the "Create an Order" command.<br/>2. User selects an existing publisher to purchase from.<br/>3. User Selects a title<br/>4. User Inputs the quantity to be ordered.<br/>5. User repeats steps 2 to 4 until all titles required for that publisher are input.<br/>6. User selects the submit button.<br/>7 Email of order is generated and sent to the Publisher.<br/>8. Order is stored locally for reference.<br/>9. Processing message is displayed to the User.<br/><br/><br/>
Order - New Publisher Alternate
Details:
1. Use case begins when User selects the "Create Order" command.<br/>2. User tries to find the publisher - no details for this publisher found.<br/>3. User requests to create a new publisher entry.<br/>4. User enters Publisher details - Name, Address, Email etc.<br/>5. Submitted details are validated.<br/>5. Processing message is displayed to the User.<br/>6. New publisher account is created in the database.<br/>7. Success message is displayed to User.<br/>8. User creates titles associated with the new publisher.<br/>9. Titles are added to the database.<br/><br/>
Order - New Title Alternate
Details:
1. User selects the "Create Order" command.<br/>2. User selects the publisher.<br/>3. User searches publisher for a known title.  Title is not found<br/>4. User presses add button and enters details of the new title.<br/>5. Submitted details are validated<br/>6. New entry for the title/publisher is created in the database<br/>7. Success message is displayed to User.<br/><br/>
