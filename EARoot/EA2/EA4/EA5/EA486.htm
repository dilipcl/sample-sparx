Scenario Type
Out of Stock on an item. Alternate
Details:
1. Use case begins when Storeroom worker selects the "Process Order" command.<br/>2. This displays the list of items under order as well as the quantity available for each of these items.<br/>3. Any Items that do not have sufficient quantity in-stock are highlighted.<br/>4. User sets the entry as out of stock.<br/>5. User checks if an order exists - if not add a stock order for the item.<br/>6. User generates an Email stating the order is awaiting stock- this is and sent to the Client.<br/>7. Processing message is displayed to the User.<br/>

Details:
